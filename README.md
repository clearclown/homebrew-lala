# homebrew-lala

Homebrew tap for [Lala](https://github.com/clearclown/lala), a modern, lightweight multi-format text editor.

## Installation

```bash
# Add the tap
brew tap clearclown/lala

# Install lala
brew install lala
```

## Upgrade

```bash
brew upgrade lala
```

## Uninstall

```bash
brew uninstall lala
brew untap clearclown/lala
```

## About Lala

Lala is a modern, lightweight text editor with GUI and CLI support for multiple formats including:
- Markdown
- HTML
- Mermaid diagrams
- LaTeX

For more information, visit the [main repository](https://github.com/clearclown/lala).
